iOS Deployment

How to install DrillAlert on your iOS device:

Send an email to lucashd@icloud.com requesting access. You will recieve an email inviting you to TestFlight for DrillAlert. Follow the directions in the email on your iOS device to install TestFlight and DrillAlert.

Open the application and log in.
Username: CAPSTONE2015\testuser
Password: StartUp!

Open the TestFlight app on your iOS device. Select DrillAlert. Select Send Feedback.

Access the code for DrillAlert for iOS:

The DrillAlert iOS App repository is availible on GitHub here: DrillAlert on GitHub
Download the project open the DrillAlert.xcodeproj file in Xcode on a Mac machine.
