Windows Deployment

How to install DrillAlert on your Windows device:

(Windows devices include any Windows 8 tablet or computer)

Send an email to tangevin@calpoly.edu requesting the application. You will recieve a folder containing the app data.

Open the folder, right click on the .ps1 file, and select run with powershell. When prompted, allow the application to be installed. The app will now be installed and can be found on your tile start screen

Open the application and log in.
Username: CAPSTONE2015\testuser
Password: StartUp!

*If there are any issues, email either tangevin@calpoly.edu or kkeim@calpoly.edu or add an issue to JIRA with details.
