Spring Dance Show Organizational Tools

For my senior project I have created a suite of tools to help organize the Cal Poly Spring Dance Show. The tools were made to perfectly fit the system used by student directors each year and should be able to be seamlessly integrated. These tools digitalize a entirely paper system and simplify tasks that take unnecessary amounts of time.

Dancer Form

The Dancer form allows dancers to submit their information by filling out an easy to use form!

Check it out! »

Choreographer Form

The choreographer form allows choreographers to submit their information and proposal in an easy to use form. Once selected they can use the submit dancers form to submit the dancer numbers they chose at auditions.

Check it out! »

Director Tool

The Director Tool is a multi-step tool that provides assitance for many different processes in the organization of the Spring Dance Show. The tool includes a to do list and access to the Dance Department Google Calendar, a tool to help select choreographers, a tool to view dancer information, generated forms for audition day, a tool to schedule rehearsal times for peices, and a tool to assign the cast.
